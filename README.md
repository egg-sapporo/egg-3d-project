# 3Dプリンター導入プロジェクト

**有限会社EGG / 劇団フルーツバスケット**

本リポジトリは、舞台小道具制作の効率化およびクオリティ向上を目的とした、3Dプリンター導入計画の提案資料を管理しています。

## プロジェクト概要

従来の「削り出し（引き算）」による小道具制作から、3Dプリンターによる「積層造形（足し算）」への移行を提案し、制作現場における課題解決（再現性の確保、修正コスト削減、脱・属人化）を目指します。

## 収録内容

- **presentation.html**
  - 導入提案プレゼンテーション資料（Webブラウザで閲覧可能）
  - キーボードの矢印キー（←/→）またはクリック・タップでスライド操作が可能です。

## 提案機材

- **Bambu Lab H2D**
  - デュアルノズル搭載により、サポート材の除去工数を劇的に削減
  - 全自動調整機能により、初心者でも安定した運用が可能

## 閲覧方法

1. リポジトリをダウンロード（またはクローン）してください。
2. `presentation.html` をお使いのWebブラウザ（Chrome, Safari, Edge等）で開いてください。
